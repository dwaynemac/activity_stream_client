= ActivityStream Client

== Usage

=== For a Rails app

In Gemfile

    gem 'activity_stream_client'

In an initializer you should define:

    ActivityStream::HYDRA
    ActivityStream::API_KEY

=== For a non-Rails app

In Gemfile

    gem 'activity_stream_client'
